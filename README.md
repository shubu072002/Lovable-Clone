## Lovable Clone – AI Full Stack App Generator

An AI-powered backend system that generates full-stack web applications using LLMs, inspired by platforms like Lovable/v0.

---

# What I Built

* Designed a scalable **Spring Boot backend** integrating LLMs for automated app generation
* Implemented **RAG + vector search (pgvector)** for intelligent context retrieval
* Reduced LLM token usage by **~50% using tool calling** → faster responses
* Built secure APIs with **JWT + OAuth2 authentication**
* Integrated **Stripe payments & webhooks**
* Used **MinIO for object storage** and **Redis for caching**

---

## ⚙️ Tech Stack

**Backend:** Java, Spring Boot, Spring Security, Hibernate (JPA)
**AI:** OpenRouter, Spring AI, RAG, Tool Calling
**Database:** PostgreSQL (pgvector), Redis
**Tools:** Docker, MinIO, Maven, Postman
**Payments:** Stripe API
